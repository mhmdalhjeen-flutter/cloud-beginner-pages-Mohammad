What is CI/CD?

CI/CD means Continuous Integration and Continuous Deployment. It is a method used in cloud systems to automatically build, test, and deploy applications after developers push changes to the repository. This helps teams deliver updates faster and reduce manual work. It also improves software quality and reliability.

My Pipeline

When I push my code to GitHub, GitHub Actions automatically runs a workflow. The workflow checks the files and then deploys the website to GitHub Pages. After the process finishes, the website becomes available online.
